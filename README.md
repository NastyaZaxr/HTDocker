ЭТОТ ПРОЕКТ СОЗДАЕТ И АНАЛИЗИРУЕТ ДАННЫЕ О КНИГАХ ПРИ ПОМОЩИ 2 DOCKER-КОНТЕЙНЕРОВ. ПЕРВЫЙ КОНТЕЙНЕР ГЕНЕРИРУЕТ СЛУЧАЙНЫЙ НАБОР ДАННЫХ О 50 КНИГАХ И СОХРАНЯЕТ ИХ В CSV-ФОРМАТЕ, А ВТОРОЙ ЧИТАЕТ, ОБРАБАТЫВАЕТ ИНФОРМАЦИЮ И СОЗДАЕТ HTML-ОТЧЕТ СО СТАТИСТИКОЙ. ДАННЫЕ ХРАНЯТЬСЯ НА ХОСТЕ - НЕ ВНУТРИ КОНТЕЙНЕРОВ.
КАК ИХ ЗАПУСТИТЬ:

'''bash'''
1)pwd - чтобы узнать полный путь к папке проекта на Вашем компьютере
2)docker run --rm -v /ПОЛНЫЙ/ПУТЬ/ДО/ПРОКЕТА/DATA:/GENERATOR/DATA generator-image python generate.py /GENERATOR/DATA - чтобы запустить генератор, после в папке DATA должен появитсья файл data.csv
3)docker buil -f Dockerfile.reporter -t reporter-image . - чтобы собрать образ аналитики
4)docker run --rm -v /ПОЛНЫЙ/ПУТЬ/ДО/ПРОКЕТА/DATA:/DATA reporter-image - чтобы появился файл report.html в папке DATA
5) ls DATA/ - првоеряем результат 
6)Откройте файл DATA/report.html в любом браузере
